### Turanova Software - Resmi Tanıtım Sitesi

## Proje Hakkında

Turanova-Software, Turanova Software markasının dijital vitrini olarak geliştirilmiş resmi tanıtım web sitesidir.

Bu proje, modern yazılım çözümlerini, geliştirilen uygulamaları ve teknik yetkinlikleri profesyonel bir arayüz ile sunmayı amaçlamaktadır.

---

## Özellikler

* Modern ve hızlı arayüz
* Responsive (mobil uyumlu) tasarım
* Yazılım projelerinin tanıtımı
* Kullanıcı dostu deneyim (UX/UI)
* Kurumsal kimlik odaklı yapı

---

## Kullanılan Teknolojiler

* HTML5
* CSS3
* JavaScript
 Kurulum
